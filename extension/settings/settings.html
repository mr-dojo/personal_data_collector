Configure your Notion integration

Create an integration at notion.so/my-integrations (Token format varies - ntn_, secret_, etc.)
Find this in your Notion database URL after the workspace name
How to set up Notion integration
  1. Go to notion.so/my-integrations
  2. Click "+ New integration"
  3. Name it "PDC Browser Extension"
  4. Copy the "Internal Integration Token"
  5. Open your PDC database in Notion
  6. Click "..." → "Add connections" → Select your integration
  7. Copy the database ID from the URL
